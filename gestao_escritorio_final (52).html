Histórico de Status

    ⚠ Dados não sincronizados

    Há alterações salvas neste computador que ainda não chegaram ao Google Sheets. Se você abrir o sistema em outro PC agora, essas alterações podem ser perdidas.

    Opção 1 — Tentar enviar ao Sheets

    Funciona se você estiver com internet. O sistema tentará enviar todos os dados agora.

    Opção 2 — Transferir para outro PC

    Se o Sheets não responder, exporte os dados locais e importe no outro PC antes de abrir lá. O arquivo gerado contém todos os dados atuais deste computador.

    O que NÃO fazer

    Não abra o sistema em outro PC enquanto este estiver com "Dados não sincronizados" — as alterações deste PC serão perdidas.

    ⚙ Google Sheets

    Cole a URL gerada ao publicar o Apps Script como Web App.

    🆘 Recuperação de emergência
    Se os dados sumirem, tente recuperar o último backup local salvo automaticamente.
